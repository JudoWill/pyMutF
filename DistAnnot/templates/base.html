This website is to facilitate the annotation of various HIV-1 mutations and their relations to interactions. A database has been created which indexes the HIV-1 Human Interaction database and the articles that it cites. I then used a regular-expression technique to extract all of the sentences which reference point mutations. These sentences are displayed to an annotator to verify the relevant information.
